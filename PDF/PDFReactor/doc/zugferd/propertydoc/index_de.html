ZUGFeRD Properties

Spalte Erklärung
Property Name Der Name des Propertys welches in der ZUGFeRD JavaScript API verwendet wird.
Beschreibung Erklärung des Elements, sowie mögliche Werte.
Benötigt für Beschreibt für welches Element dieses Element angegeben werden muss, damit dieses Element generiert wird. Steht in diesem Feld "ZUGFeRD", so kann ohne dieses Property kein ZUGFeRD PDF erzeugt werden.
Profil Das ZUGFeRD Profil in welchem das ZUGFeRD Property auftritt. Wird in der ZUGFeRD JavaScript API das Profil "basic" angegeben, so werden keine Elemente mit dem Profil "comfort" erzeugt. Im Profil comfort sind jedoch ZUGFeRD Properties des Profils basic erlaubt.
Typ Der Typ des ZUGFeRD Propertys. Je nach Typ muss der zu setzende Wert bestimmte Voraussetzungen erfüllen.

Property Typen

Text

Der Typ Text beschreibt einen Freitext, bzw. eine Textangabe die nicht validiert wird.

Number

Der Typ Number beinhaltet Beträge sowie Prozentwerte. Das Dezimaltrennzeichen muss über das inputFormat Property angegeben werden, um die Validität des ZUGFeRD XML zu gewährleisten. Genauere Information zum inputFormat Property sind in der API Dokumentation zu finden.

Date

Ein Datum wird mit dem Typ Date gekennzeichnet. Der Date Typ benötigt das aktuelle Eingabeformat für das Datum, angegeben über das inputFormat Property, sowie das Ausgabeformat welches im ZUGFeRD eingetragen wird, angegeben über das scheme Property. Genauere Information zu diesen Properties sind in der API Dokumentation zu finden. Mögliche Ausgabeformate sind:

102Jahr, Monat und Tag: yyyyMMdd
610Jahr und Monat: yyyyMM
616Jahr und Kalenderwoche: yyyyww

Document Type

Der Document Type beschreibt die Art des Dokuments mit einem Code. Mögliche Dokumentarten sind:

84Wertbelastung
380Handelsrechnung
389Selbst ausgestellte Rechnung

Subject Code

Der Subject Code beschreibt den Inhalt einer Notiz, um diese elektronisch prüfen zu können. Mögliche Codes sind:

""Allgemeine Freitexte
REGRegulatorische Informationen
AAKPreiskonditionen
AAJZusätzliche Konditionen zu diesem Kauf
PMTZahlungsinformation

Global ID

Der Typ Global ID gibt die Organisation an, in welcher die ID des Käufers, Verkäufers oder des Produkts, registriert ist. Der Code dieser Organisation muss über das Property scheme angegeben werden. Genauere Information zum diesem Property sind in der API Dokumentation zu finden. ZUGFeRD gibt folgende Beispiele:

0021SWIFT (BIC)
0060D-U-N-S Number
0088GS1 Global Location Number (GLN)
0160GS1 Global Trade Item Number (GTIN, EAN)
0177OSCAR (Odette)
