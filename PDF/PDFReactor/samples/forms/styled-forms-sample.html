Demonstration of non-interactive PDF forms and AcroForms

This document demonstrates the HTML forms and interactive PDF forms (sometimes refered to as AcroForms) capabilities of PDFreactor. Each type of each form element will be displayed, at first as a non-interactive PDF form, then as an AcroForm. Additionally, these categories are repeated as CSS styled forms. At last, the auto resizing feature is demonstrated for input elements of the type "text" and textarea elements. For more information about resizing form controls, please have a look at chapter "Automatic Resizing of Form Controls" in the PDFreactor manual.
